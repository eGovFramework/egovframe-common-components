# egovframe-common-components
 252 common functions that are reusable.

## Requirements
* JDK 1.8 or JDK 11
* Maven 3.1+

## Build
```
mvn clean package
```

## SQL
1. Check the database connection.
2. Run [ddl script](./script/ddl/)
2. Run [comment script](./script/comment/)
3. Run [dml script](./script/dml/)
